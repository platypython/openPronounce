Contribute a new pronunciation

Add a new folder under /projects. The site discovers folders automatically via the GitHub Contents API. 

/projects/<project-name>/
  <project-name>.mp3
  description.txt
  icon.png (optional)
  • MP3 filename: Prefer <folder>.mp3. If you upload multiple MP3s, the UI will pick one (best effort).
  • Description: Put plain text in description.txt.
  • Icon: Optional icon.png. Keep it under 128×128 (and ideally < 50KB). The UI will display all icons at the same size.

Built for GitHub Pages. No build step. No frameworks.
